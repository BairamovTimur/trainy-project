## Установка

- Устанавливает зависимости в проекте

для корректной работы требуется node.js версия не ниже 20.18.3

```sh
npm install
```

## Разработка

- Запускает приложение в режиме разработки. Веб-сервер будет доступен по адресу [http://localhost:3000](http://localhost:3000)

```sh
npm start
```

- Создает сборку проекта для продакшн-окружения.

```sh
npm build
```

## Тестирование

- Запускает и выполняет все юнит-тесты проекта без покрытия

```sh
npm test
```

- Запуск только определенного файла с тестами

```sh
npm test src/__tests__/ваш_путь
```

- Запускает и выполняет все юнит-тесты проекта с покрытием

```sh
npm test_coverage
```

- Запуск только определенного файла с тестами и покрытием

```sh
npm test_coverage src/__tests__/ваш_путь
```

Задача №1
1. Структура страницы:
 • Главная страница состоит из 4 одинаковых блоков, расположенных в виде сетки 2 на 2 (то есть 2 строки и 2 столбца). • Между блоками должно быть одинаковое расстояние – 16 пикселей. • Каждый блок окружён чёрной рамкой (границей) толщиной 3 пикселя.
2. Функциональность блоков:
 • Каждый блок служит ссылкой на другую страницу приложения. • При клике на блок пользователь переходит на соответствующую страницу. • На данном этапе эти страницы могут быть заглушками – достаточно отобразить их название. • Ссылка внутри блока состоит из текста и иконки. Для иконок используйте библиотеку antd (Ant Design Icons).
3. Адаптивность:
 • Все блоки и их содержимое должны корректно изменять размеры при изменении разрешения экрана или масштабирования, сохраняя пропорции и отступы.
4. Названия блоков:
 • Верхний левый блок: «Счетчик» • Верхний правый блок: «Продвинутый счетчик» • Нижний левый блок: «Список сотрудников» • Нижний правый блок: «Таблица»
5. Навигация:
 • Для переходов между страницами следует использовать возможности библиотеки react-router (она уже подключена в проект).
6. Визуальное оформление:
 • Оформление (цвета, шрифты, отступы внутри блоков) определяется по вашему усмотрению, главное – сделать дизайн современным и привлекательным.

Схематическая визуализация главной страницы
Представьте следующую простую схему (каждый блок обозначен как прямоугольник с названием):
         ┌───────────────────────┐    ┌───────────────────────┐
         │      СЧЁТЧИК         │    │  ПРОДВИНУТЫЙ СЧЁТЧИК   │
         │ [иконка + текст ссылки]│    │ [иконка + текст ссылки]│
         └───────────────────────┘    └───────────────────────┘
                     16px между блоками
         ┌───────────────────────┐    ┌───────────────────────┐
         │   СПИСОК СОТРУДНИКОВ  │    │       ТАБЛИЦА         │
         │ [иконка + текст ссылки]│    │ [иконка + текст ссылки]│
         └───────────────────────┘    └───────────────────────┘
Каждый из этих блоков имеет:
 – Рамку вокруг себя (3px чёрная линия)
 – Отступы между блоками по 16px
 – Адаптивный размер, который подстраивается под размер экрана
──────────────────────────────
Итог:
Таким образом, задание предусматривает создание адаптивной главной страницы с 4 блоками, каждый из которых является ссылкой (с иконкой и текстом) на отдельную страницу. Реализуйте переходы с помощью react-router, а иконки берите из библиотеки antd. Дополнительные стилистические решения остаются на ваше усмотрение, главное – добиться аккуратного и современного внешнего вида.
